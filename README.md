# Serendipity for APP
Serendipity theme adapted to APP


# Instructions

- Open Settings
- Select Display from Sidebar
- Select Theme > Edit
- Select Custom Themes
- Copy the content from .json files


# Available themes
- Midnight
- Sunset
- Morning
